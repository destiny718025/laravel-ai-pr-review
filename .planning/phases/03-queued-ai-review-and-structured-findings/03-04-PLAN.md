| T-03-10 | Tampering | `review_findings` replacement writes | high | mitigate | Replace the full finding set transactionally through `ReviewFindingRepository::replaceForReviewRun()` so retries cannot mix old and new findings. |
| T-03-11 | Information Disclosure | `review_runs.safe_error_message` and findings UI | high | mitigate | Persist only whitelisted safe summaries, and render persisted findings fields only; never display raw provider payloads or credentials per D-15. |
| T-03-12 | Elevation of Privilege | Phase 3 detail page actions | medium | mitigate | Keep the detail page limited to run/retry and findings display; do not add draft approval or GitHub publication controls reserved for later phases. |
| T-03-SC | Tampering | npm/pip/cargo installs | low | accept | No package-manager installs are introduced in this plan; migration and persistence use existing Laravel features only. |
</threat_model>
