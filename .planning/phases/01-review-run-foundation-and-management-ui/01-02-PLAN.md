<objective>
Create the parser, DTOs, repositories, and service workflow that turn a GitHub PR URL into a pending review run.

Purpose: Establish Controller / Service / Repository creation boundaries before routes and views call into them.
Output: URL parser, DTOs, repository classes, ReviewRunService, and focused creation/failure tests.
</objective>
